### 프로필 생성하기
```
- Github 접속 > Repository > New 클릭하여 신규 repository를 생성해준 뒤 이름을 본인의 username과 동일하게 입력한다.
- 생성된 README.md 파일에서 github 프로필 작성 가능하다.
```
